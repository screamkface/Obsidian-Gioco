# Eye Safe Writer

Warm, low-distraction writing environment with soft paper texture, calm panels, and palette-aware reading colors.

## Long-form writing

This theme is tuned for daily notes, essays, worldbuilding, research, and long writing sessions. Text remains readable, links stay clear, and UI chrome fades back while the note stays centered.

> [!note]
> Use this note for a light-mode screenshot. Recommended setup: light mode, Salvia palette, paper grain texture, comfortable UI density.

### Formatting sample

Writing can include **strong emphasis**, *quiet emphasis*, ==highlighted phrases==, `inline code`, [[Internal Links]], [external links](https://obsidian.md), and #tags without breaking the calm visual tone.

```markdown
The theme keeps editor mode and reading mode aligned with the selected palette.
```
