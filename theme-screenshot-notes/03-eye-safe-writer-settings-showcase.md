# Style Settings Showcase

Use this note while opening Style Settings beside it. Capture theme customization controls if you want a README image showing configurability.

## Recommended controls to show

1. Palette selector.
2. Heading color style.
3. Link style.
4. UI density.
5. Panel style.
6. Texture type.
7. Grain amount.
8. Writing focus intensity.

> [!info]
> Settings dropdowns should show a single arrow only. If duplicated arrows appear, reload theme CSS and retest.

## Texture examples

- Paper grain for classic warm writing.
- Fine dust for minimal texture.
- Linen weave for tactile surface.
- Speckles for analog paper feel.
- Diagonal fibers for visible paper direction.
- None for maximum contrast and performance.
