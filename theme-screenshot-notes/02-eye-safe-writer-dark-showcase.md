# Eye Safe Writer Dark

Dark palettes keep the same writing-first layout while adapting surfaces, accents, code, tables, callouts, and UI controls.

> [!tip]
> Use this note for a dark-mode screenshot. Recommended setup: dark mode, Gruvbox or AMOLED palette, fine dust texture, deep focus mode.

## Palette consistency

- Links stay readable without visual noise.
- Callouts inherit Obsidian callout colors while using theme surfaces.
- Tables keep quiet borders for dense notes.
